# 变更日志

## [1.1.0] - 2026-03-14

### 新增
- 权限管理系统
  - 新增 guest 访客角色，只能查看项目菜单
  - 路由级别权限控制，防止越权访问
- 用户账号
  - guest / guest123（访客，只读权限）
- 首页
  - 登录后默认进入首页
  - admin 用户可通过首页或顶部菜单访问菜单管理和系统设置

### 修改
- admin 角色保持完整管理权限
- guest 用户无法访问菜单编辑器和系统设置页面
- 侧边栏只显示项目菜单，移除系统菜单（菜单管理、系统设置）
- admin 用户通过首页快捷按钮访问菜单管理、系统设置
- 移除头像下拉菜单，点击头像直接触发退出登录确认
- 移除登录页默认账号密码提示
- 优化菜单配置结构，每个项目及子项目包含：规格说明书、预览链接、变更日志
- 移除首页帮助气泡和访客提示信息

### 删除
- 移除仪表盘页面

### 修复
- 修复 guest 登录后跳转到 404 页面的问题（登录后默认跳转到首页）
- 修复 404 页面返回按钮跳转路径错误
- 修复菜单配置缓存导致 guest 仍能看到系统菜单的问题（添加版本检查机制）

## [1.0.0] - 2026-03-13

### 新增
- 项目初始化，创建基础目录结构
- 实现 Vue 3 + Vite 5 + Element Plus 2 技术栈
- 实现用户认证模块（简单登录）
- 实现主布局组件（侧边栏、顶部导航）
- 实现仪表盘页面（统计卡片、快速入口）
- 实现可视化菜单编辑器
  - 分组管理（添加、编辑、删除、拖拽排序）
  - 菜单项管理（添加、编辑、删除、拖拽排序）
  - 实时预览
  - 配置导出
- 实现项目列表页面
- 实现内容查看页面
  - iframe 查看器（Web 应用）
  - 移动端模拟器（iPhone 外观）
  - Markdown 文档查看器
- 实现系统设置页面
  - 主题切换
  - 菜单配置管理
- 实现帮助气泡组件
- 实现 404 错误页面

### 技术特性
- 采用苹果设计风格
- CSS 变量实现主题切换
- Pinia 状态管理
- Vue Router 路由守卫
- vuedraggable 拖拽排序
- marked + highlight.js Markdown 渲染
